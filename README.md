# tracys-site
Tracy's Hair Studio

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd <project-name>
```

2. Install all project dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Create a production build:

```bash
npm run build
```

5. Preview the production build:

```bash
npm run preview
```
